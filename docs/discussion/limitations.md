# 局限性与未来工作

## 局限性

虽然在MuJoCo Humanoid仿真数据上进行了验证，但与真机部署仍存在一定差距。

SSM-WM在合成数据集上的预测精度低于LSTM-WM (差距约167%)，但在MuJoCo数据集上反而优于LSTM (差距约6%)。在推理速度方面，SSM-WM在高维状态空间 (376维) 下的推理时间 (9.5ms) 高于LSTM (5.0ms)，主要受FFT卷积在高维状态下的开销影响。

SSM-WM在MuJoCo数据集上的R²分数为0.592，约41%的方差未被解释，对于需要精确预测接触力的激进操作任务可能精度不足。

当前模型仅处理状态-动作序列，未涉及视觉输入，限制了其在端到端控制中的应用。

## 失败模式

SSM-WM在以下场景可能表现不佳:

- **极短序列** (T<16): 无法充分利用长程依赖，预测精度显著下降
- **高度不连续动力学**: SSM的线性状态转移假设在剧烈碰撞场景下可能失效
- **分布外泛化**: 训练与测试数据分布差异大时精度急剧下降

## 未来工作

1. **视觉输入扩展** — 结合CNN或ViT进行视觉状态预测
2. **真机验证** — 在MuJoCo物理仿真器和真机人形机器人上实验
3. **嵌入式部署** — NVIDIA Jetson上推理时间和能效分析
4. **多模态融合** — 视觉、触觉和本体感觉
5. **结构化稀疏参数化** — 利用关节耦合拓扑设计特定的A矩阵结构
6. **强化学习结合** — 端到端感知-决策-控制一体化框架
