# 消融实验

## 架构消融 (表5, 合成数据集)

| 配置 | MSE | 参数 (M) | 时间 (ms) |
|------|-----|----------|-----------|
| 完整SSM-WM | 2.72 | 0.24 | 3.8 |
| 去除门控 | 2.78 | 0.22 | 3.5 |
| 去除残差 | 2.76 | 0.24 | 3.8 |
| L=2层 | 2.99 | 0.12 | 2.9 |
| L=6层 | 2.64 | 0.36 | 4.7 |
| N=32 | 2.68 | 0.25 | 3.9 |
| D=64 | 2.93 | 0.08 | 2.1 |
| D=256 | 2.58 | 0.85 | 6.8 |

**门控贡献2.2%，残差贡献1.5%，D=128是性能饱和点。**

## 训练损失消融 (表6)

| 配置 | 单步MSE | 多步MSE (H=8) |
|------|---------|---------------|
| λ=0 (仅单步) | 2.64 | 9.99 |
| λ=0.1 | 2.68 | 8.07 |
| λ=0.5, H=4 | 2.74 | 7.52 |
| **λ=0.5, H=8** | **2.72** | **7.17** |
| λ=1.0 | 2.84 | 7.05 |

多步损失对多步精度贡献显著 (降28.2%)，对单步精度影响较小。

## 门控阈值对比 (表10, MuJoCo)

| 阈值函数 | MSE | R² |
|----------|-----|-----|
| **软阈值 σ(x)** | **0.834** | **0.592** |
| Garrote | 0.852 | 0.583 |
| 硬阈值 | 0.891 | 0.564 |

软阈值比硬阈值好6.4%，比Garrote好2.1%。三种阈值推理时间无显著差异。

## MuJoCo消融 (表9)

| 配置 | MSE | R² |
|------|-----|-----|
| 完整SSM-WM | 0.834 | 0.592 |
| 去除门控 | 0.872 | 0.573 |
| 去除残差 | 0.856 | 0.581 |
| L=2层 | 0.912 | 0.554 |

各组件在MuJoCo上的贡献比合成数据集更显著，说明架构设计对高维非线性建模更关键。
