# 实验设置

## 数据集

**合成数据集**: 500条episode，每条约200步，状态维度376，动作维度17。每条episode具有不同的随机动力学参数，模拟不同任务场景。400条训练，100条验证。

**MuJoCo Humanoid**: 使用MuJoCo物理仿真器的Humanoid-v4环境，200条episode，状态维度376（关节角度、角速度、身体姿态），动作维度17（关节力矩），包含真实接触动力学、摩擦力和碰撞响应。

## 基线方法

| 方法 | 说明 | 参数量 |
|------|------|--------|
| LSTM-WM (2层) | 2层LSTM，隐层128 | 0.29M |
| LSTM-WM (4层) | 4层LSTM，隐层128 | 0.38M |
| Transformer-WM | 标准Transformer，隐层128 | 0.62M |
| Mamba-WM | 官方mamba-ssm库实现 | 0.28M |
| SSM-WM (本文) | S4D对角SSM + Mamba门控 | 0.24M |

## 评价指标

- **均方误差 (MSE)**: $\times 10^{-3}$
- **决定系数 (R²)**: 模型解释方差比例
- **推理时间**: 单批次前向传播时间 (ms)
- **参数量**: 可训练参数数量 (M)

## 实现细节

所有模型训练20个epoch，使用AdamW优化器，初始学习率 $1\times10^{-4}$，余弦退火调度，梯度裁剪阈值1.0。推理时间在卷积模式下测量，批次大小64。每组实验5个随机种子。
